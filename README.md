# Deviation Analysis

This project is designed for analyzing deviations in data.

## Setup

1. Create virtual environment:
```bash
python -m venv deviation_env
```

2. Activate virtual environment:
- Windows:
```powershell
.\deviation_env\Scripts\Activate
```
- Unix/MacOS:
```bash
source deviation_env/bin/activate
```

3. Install dependencies:
```bash
pip install -r requirements.txt
```

## Project Structure

- `src/`: Source code
- `tests/`: Test files
- `requirements.txt`: Project dependencies
